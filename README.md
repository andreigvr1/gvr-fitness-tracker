# GVR Fitness Tracker

PWA de antrenament: genereaza programe personalizate (obiective, zile, timp, echipament,
articulatii sensibile) si ghideaza progresia sesiune de sesiune. Include modul dedicat
de skandenberg/armwrestling cu adaptare pe stil si echipament.

- Specificatia completa: `docs/specificatie_mvp.md`
- Instructiunile de proiect pentru Claude Code: `CLAUDE.md`
- Istoric versiuni: `CHANGELOG.md`

Aplicatia ruleaza ca PWA pe GitHub Pages, offline, cu datele stocate local pe telefon.
