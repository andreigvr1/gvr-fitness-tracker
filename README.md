# GVR Fitness Tracker

PWA de antrenament: genereaza programe personalizate (gen, obiectiv, zile, timp, experienta,
echipament, articulatii sensibile) si ghideaza progresia sesiune de sesiune. Modulul dedicat
de skandenberg/armwrestling este planificat post-MVP (continutul exista deja in biblioteca).

Aplicatia live: https://andreigvr1.github.io/gvr-fitness-tracker/
